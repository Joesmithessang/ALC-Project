About LagDevs

Purpose Of App

The Lagos Devs application list developers in Lagos who uses java programming language as their main development language. The data are queried from the public github Api using the url https://api.github.com/search/users?q=location:lagos+language:java
It was developed as a pre-requisite for progression at Andela Learning Community.
